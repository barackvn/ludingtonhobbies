POS Stock

Features :
  • Display of current stock of products in a running session.
  • Select stock type for display product stock in POS.
  • Seller can deny or allow order for negative stock.
  • Seller can set the stock limit below which order will be denied.
  • Seller can set a custom error message for out of stock products.
